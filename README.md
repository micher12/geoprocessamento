# Projeto Geoprocessamento
<p>Trata-se de um projeto da faculade, tem como objetivo ordenar 100 mil imagens sobre o ambiente do serrado. Com isso surgiu este software que recebe uma imagem anterior e posterior de alguma ação humana que prejudicou o local</p>

<br/>

### > Listagem +100 mil dados e +200 mil imagens.
<p>• São +200 mil imagens pois, ao clicar em abrir imagem  é mostrado a imagem anterior e posterior de alguma mudança detectada no ambiente.</p>
